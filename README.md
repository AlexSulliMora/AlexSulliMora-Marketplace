# AlexSulliMora-Marketplace

Personal Claude Code plugin marketplace.

## Plugins

- **[coauthor](plugins/coauthor/README.md)** — five-stage research workflow (`/scope`, `/plan`, `/work`, `/review`, `/finalize`) with named worker agents (analyst, coder, writer, reviewer, researcher), per-project document artifacts, three-tier audit logging, and a promotable validator library.

## Install

```
/plugin marketplace add AlexSulliMora/AlexSulliMora-Marketplace
/plugin install coauthor@AlexSulliMora-Marketplace
```

See the [coauthor README](plugins/coauthor/README.md) for usage, stages, configuration, and customization.
